# Project Glossary

> **Last Updated:** 2026-06-16

---

| Term | Definition |
|------|-----------|
| **WBS** | Work Breakdown Structure — hierarchical decomposition of project scope into deliverable phases |
| **PMO** | Project Management Office — governance body overseeing all projects |
| **HOD** | Head of Department — senior management approver |
| **SPM** | Senior Project Manager — oversees multiple PMs and projects |
| **EM** | Engagement Manager — manages client relationships and commercial agreements |
| **PM** | Project Manager — day-to-day project execution owner |
| **TL** | Team Lead — technical lead managing developer team |
| **BO** | Business Owner — executive with portfolio-level oversight |
| **Dhanshree** | Delivery Operations role — super-admin with full system access |
| **Pulse PMO** | Internal brand name for the Project Compass platform |
| **Health Status** | Project risk indicator: Green (healthy), Amber (at risk), Red (critical) |
| **Bucket** | PM capacity allocation tracker (percentage utilized) |
| **Bench** | Resources not currently assigned to any project |
| **Shadow Team** | Backup/support resources assigned to a project but not primary |
| **Fit Score** | Algorithm-calculated match percentage for resource allocation |
| **Prerequisite** | Documents/validations required before project kickoff |
| **Stage Tracker** | Visual pipeline: Sales → PMO → Delivery → Accounts |
| **PO** | Purchase Order — client's formal order document |
| **SOW** | Statement of Work — project scope document |
| **CR** | Change Request — formal scope modification |
| **UAT** | User Acceptance Testing — client validation phase |
| **Steerco** | Steering Committee — cross-functional governance meeting |
| **Utilization** | Percentage of a resource's capacity currently allocated |
| **Billable** | Work that can be charged to the client |
| **Non-Billable** | Internal/overhead work not charged to client |
| **Fixed Resource** | Resource committed for the project duration |
| **Adhoc Resource** | Resource assigned on an as-needed basis |
| **Cell Comment** | Per-day comment on a timesheet entry for clarification |
| **Allocation Event** | Record of resource assignment/reassignment |
| **Escalation Chain** | Hierarchy for issue elevation: TL → PM → SPM/EM → PMO/HOD |
